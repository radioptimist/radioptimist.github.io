## Controls for PyVista Plots throughout this site
Some basic controls for the interative plots on this site:
 - Rotate using left click and drag
 - Pan using left click while holding shift
 - Zoom using the mouse wheel
 - Reset pan/zoom by pressing R key
 - Show the wire edges of a mesh with the W key
 - Show the vertices of a mesh using the V key
 - Return to the full mesh using the S key
